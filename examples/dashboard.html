Perguntas Frequentes

Como posso integrar esse dashboard com meu sistema?

O dashboard foi criado com HTML5 semântico e CSS puro, facilitando a integração com qualquer framework ou CMS. Você pode customizar cores, fontes e componentes diretamente no CSS.

Qual é a melhor forma de usar este framework?

Recomendamos usar o arquivo style.css como base para todos os seus projetos. Você pode estender com arquivos CSS específicos para cada página, como fazemos com dashboard-custom.css.

O framework é responsivo?

Sim! Todos os componentes foram desenvolvidos com mobile-first em mente. Utilize media queries para ajustes em diferentes tamanhos de tela.
